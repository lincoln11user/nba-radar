# NBA 球员雷达图 🏀

> 12 轴能力池 · 位置模板 10 轴适配 · 单人/1v1/多人/历史赛季对比

一个基于 **ECharts** 的 NBA 球员能力可视化工具。选择球员，看他的技术雷达图一目了然。

## 功能亮点

### 📊 12 轴能力池
三分 · 中距离 · 内线终结 · 罚球 · 外线单防 · 护框 · 协防换防 · 抢断压迫 · 组织 · 控球 · 篮板 · 运动能力

### 🎯 位置模板自动适配
根据球员位置（PG/SG/SF/PF/C），自动从 12 轴中选出最相关的 **10 轴**展示。跨位置比较时自动切换为 12 轴全量对比。

### 🔄 三种对比模式
| 模式 | 说明 |
|------|------|
| **单人** | 看一个球员的全面能力 |
| **1v1** | 两名球员直接 PK（同位置/跨位置） |
| **多人** | 最多 4 名球员一起对比 |

### 📋 赛季数据展示
得分、助攻、篮板、命中率等关键数据，附能力值可视化条形图。

### 🏷️ 筛选与分类
按位置（PG→C）、按时代（现役/传奇）过滤球员。

## 快速开始

```bash
# 直接用浏览器打开
open index.html

# 或用任意 HTTP 服务器
python -m http.server 8080 --directory .
```

> 因为用了 `fetch('players.json')` 加载数据，**需要用 HTTP 服务器打开**，不能在 `file://` 下运行。

## 数据结构

`players.json` 包含每个球员的：
- 基本信息（姓名、位置、球队）
- 12 项能力评分（0-100）
- 赛季统计（场均得分、篮板、助攻等）
- 头像链接

## 技术栈

- [ECharts 5.5.1](https://echarts.apache.org/) — 雷达图渲染
- 纯原生 HTML/CSS/JS — 无框架依赖
- 响应式布局 — 桌面/移动端均可用
